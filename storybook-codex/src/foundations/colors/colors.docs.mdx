# Colors

The color system defines brand, semantic meaning, and mode-aware behavior.

## Philosophy

Use semantic tokens in components so color mode can switch without component-level overrides.

## Accessibility

- Verify text contrast in both modes.
- Do not rely on color alone for status meaning.
- Keep state feedback visible beyond color changes.
