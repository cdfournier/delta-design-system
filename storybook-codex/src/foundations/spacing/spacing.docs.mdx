# Spacing

Consistent spacing creates visual rhythm, improves readability, and establishes clear relationships between elements.

## Spacing Philosophy

The spacing scale adapts responsively.
Mobile/tablet uses a compact scale and widescreen doubles values to match larger typography and layouts.

## Accessibility

### Touch targets

- Minimum touch target: 44x44px
- Recommended touch target: 48x48px
- Minimum spacing between targets: `md` in mobile mode

### Visual separation

- Related elements: `sm` to `md`
- Distinct sections: `xxxl` to `xxxxxl`
- Adequate white space reduces cognitive load

### Vertical rhythm

Use consistent paragraph/list/heading spacing tokens to maintain predictable reading flow.

### Horizontal spacing

Use `md`/`lg` for related gaps, and larger tokens for stronger separation.

## Usage Guidelines

### Do

- Use spacing tokens instead of raw one-off values
- Let responsive scaling handle mode differences
- Scale spacing with content density and hierarchy

### Don't

- Do not invent arbitrary values (for example 13px)
- Do not disable responsive spacing behavior
- Do not skip existing scale steps
