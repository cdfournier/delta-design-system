# Icons

Icons are 24x24 visual symbols used to support navigation, actions, and status in the Delta Design System.

## Anatomy

- Stroke: 1px line weight
- Frame: 24px x 24px (`1.5rem x 1.5rem`)
- Color: inherits `text/default`

## Accessibility

- Use `aria-hidden="true"` for decorative icons.
- Use `role="img"` + `aria-label` when icon meaning is required.
- Keep icon-only interactive elements at least 44x44px touch target.
- Do not rely on icon shape alone for critical meaning.

## Usage

### Do

- Use the same icon name for the same action across components.
- Pair icons with labels/tooltips when meaning is not obvious.
- Keep stroke and frame consistent.

### Don’t

- Do not mix icon sets with different styles.
- Do not change stroke width per icon.
- Do not use icons as the only indicator for errors or warnings.

## Source

The icon set is aligned with [Tabler Icons](https://tabler.io/icons) naming and path style.
