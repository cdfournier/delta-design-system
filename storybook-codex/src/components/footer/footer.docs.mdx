# Footer

Footers are organism-level containers for global navigation, branding, and utility controls.

## Anatomy

- Outer wrapper (`footer`): spacing shell
- Group container (`group`): section background and rounded surface
- Navigation (`nav > ul > li > link`): list of small bold links
- Brand block: Delta mark + heading
- Color mode row: sun icon, switch, moon icon
- Fine text: legal/copyright line

## Usage

### Do

- Keep links concise and scannable.
- Use one consistent heading for the brand block.
- Reuse existing link/switch/icon atoms for parity.

### Don't

- Do not mix unrelated utility controls into this component.
- Do not hardcode spacing outside tokenized values.
