# Pagination

Pagination provides visual navigation for carousels, slideshows, and multi-step flows.

## Anatomy

- Nav container: semantic wrapper with `aria-label`
- Dot list: rounded pill background with horizontal spacing
- Dot buttons: circular indicators with selected/unselected states

## Accessibility

- Wrap in `<nav aria-label="Pagination">`.
- Use `<button>` for each dot and `aria-current="page"` on selected item.
- Keep keyboard focus visible for interactive dots.
