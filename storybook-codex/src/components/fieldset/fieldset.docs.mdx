# Fieldsets

Fieldsets wrap related input controls into consistent form groups with labels/legends, state styling, and helper messaging.

## Anatomy

- Label or legend: contextual heading for the grouped control(s)
- Input body: text/password/select control or a 4-item choice list
- Optional legend label: secondary descriptor in choice groups
- Optional message: helper/validation copy below the control(s)

## Controls

Use `type` to switch fieldset composition, then configure the embedded input variant controls:

- `state` for text/password/select state coverage
- `checkboxPattern`, `radioSelection`, `switchPattern`, and `direction` for choice groups
- `showLegendLabel`, `showMessage`, and `messageState` for structural/content options
