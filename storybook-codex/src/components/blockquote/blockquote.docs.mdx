# Blockquotes

Blockquotes highlight quotations from external sources, adding visual emphasis and proper semantic structure to cited content.

## Anatomy

Blockquotes consist of quoted text and an optional citation.

- Border: `1px` solid `global/gray-medium`
- Quote text: bold quote content with responsive scaling
- Citation: attribution text that can include a source URL via `cite`
- Shape: radius token `border-radius/sm` (`8px / 0.5rem`)

## Variants

Blockquote uses one core style, with contextual background options:

- `page`
- `section`
- `transparent`

## States

Blockquotes are non-interactive content elements and have no hover/focus/pressed states.

## Accessibility

### Semantic HTML

Use native `<blockquote>` and `<cite>` for quoted content and attribution.

### Citations

Use the `cite` attribute when source URLs are available.

### Content

Do not use blockquotes for visual indentation of non-quoted text.

## Usage

### When to use

- External quotes and references
- Testimonials
- Pull quotes for key passages

### When not to use

- Pure indentation or decoration
- Headings or non-quoted body content
- Inline quotes (use `<q>` for short inline quotations)

## Usage Guidelines

### Do

- Use semantic quote markup
- Preserve quote accuracy
- Include attribution when possible
- Place quotes in natural reading flow

### Don't

- Do not misuse blockquote as layout indentation
- Do not alter source wording without clear ellipsis usage
- Do not omit source context when attribution is needed
