# Badges

Badges are small status indicators that draw attention to new, important, or categorized content.
They provide quick visual feedback about an item's state or classification.

## Anatomy

Badges consist of:

- a container
- text content
- rounded corners

They are compact and designed for quick scanning.

## Variants

The Delta Design System provides three badge variants:

- `primary`: emphasis or featured content
- `secondary`: general information or categorization
- `transparent`: subtle emphasis or alternative styling

## States

Badges are static informational elements.
They do not have interactive states like hover, focus, or pressed.

## Accessibility

### Color and meaning

Do not rely on color alone to convey meaning.
Badge text should communicate the status/category without needing color.

### Contrast requirements

Badge foreground/background combinations should meet WCAG AA contrast requirements.

### Screen readers

Use descriptive text.
Avoid vague labels that require visual context to be understood.

## Usage

### When to use

- Status indicators: show item state (active, pending, complete)
- Categorization: tag content by type or topic
- Counts: display numerical information (for example unread messages)
- Highlights: draw attention to new or important items

### When not to use

- Interactive elements: use buttons or chips for clickable behavior
- Long text: badges should stay short and scannable
- Primary actions: badges are informational, not calls to action

### Content guidelines

- Keep text short: one to two words or short numbers
- Use sentence case
- Use consistent color semantics across the interface

### Placement

- Inline with text when attached to nearby content
- Near the top-right of cards/items when indicating status
- In groups when multiple categories/statuses are needed

## Usage Guidelines

### Do

- Keep text concise
- Use consistent color semantics
- Provide enough text context for non-visual users

### Don't

- Do not use badges as interactive controls
- Do not use long phrases in badges
- Do not rely on color alone
