# Tables

Tables display structured data in rows and columns for scanning and comparison.

## Anatomy

- `table-cell`: body cell text
- `table-cell (header)`: header cell text
- `table-cell (control)`: checkbox control cell
- `thead`: header row group
- `tbody`: body row group
- `table`: assembled component

## Accessibility

- Uses semantic `<table>`, `<thead>`, `<tbody>`, `<th scope="col">`, and `<td>`.
- Overflow wrapper is keyboard-focusable and horizontally scrollable.
- Control cells use native checkbox inputs with labels.
