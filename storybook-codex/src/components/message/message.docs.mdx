# Messages

Messages provide modal, promo, inline status, confirmation, and toast feedback patterns.

## Anatomy

- Container: variant background + spacing shell
- Message content: either Content composition or concise status text
- Actions: confirmation Yes/No buttons where required
- Dismiss control: close icon button (when enabled)
